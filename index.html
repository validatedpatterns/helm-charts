Helm Charts for Validated Patterns

Usage


        helm repo add validated-patterns https://charts.validatedpatterns.io

These is the list of charts used by the Validated Patterns project. For more information consult the Validated Patterns website.

Charts

  • aap-config ( 0.1.4 @ )

    A Helm chart to build and deploy secrets using external-secrets for ansible-edge-gitops

  • acm ( 0.1.7 @ )

    A Helm chart to configure Advanced Cluster Manager for OpenShift.

  • ansible-automation-platform-instance ( 0.1.1 @ )

    A Helm chart to install an ansible-automation-platform instance

  • clustergroup ( 0.9.30 @ )

    A Helm chart to create per-clustergroup ArgoCD applications and any required namespaces or subscriptions.

  • edge-gitops-vms ( 0.2.9 @ )

    Edge GitOps VMs

  • ensure-openshift-console-plugins ( 0.1.1 @ )

    A Helm chart to ensure openshift console plugins

  • external-dns-operator ( 0.1.2 @ )

    External DNS Operator chart

  • gitea ( 0.0.3 @ 1.22.6 )

    Gitea Helm chart for Kubernetes adapted for VP

  • golang-external-secrets ( 0.1.4 @ )

    A Helm chart to configure the golang-based external-secrets.

  • hashicorp-vault ( 0.1.5 @ )

    A Helm chart to configure Hashicorp's vault.

  • letsencrypt ( 0.1.1 @ 1.16.0 )

    A Helm chart to add letsencrypt support to Validated Patterns.

  • node-health-check-operator ( 0.1.0 @ )

    Chart to configure node-health-check-operator

  • openshift-data-foundations ( 0.2.0 @ 1.16.0 )

    A Helm chart to install ODF on Openshift

  • openshift-virtualization-instance ( 0.1.0 @ )

    OpenShift Virtualization

  • pattern-install ( 0.0.8 @ )

    A Helm chart to build and deploy a Cloud Pattern via the patterns operator

  • rbac ( 0.1.0 @ 1.16.0 )

    A Helm chart that provisions service accounts, roles and rolebindings

  • servicemesh ( 0.1.0 @ 3.0.0 )

    Helm chart for deploying a fully functional Red Hat OpenShift Service Mesh 3.0 base setup, including support for sidecar and ambient modes, and monitoring integration.

  • vp-rbac ( 0.1.2 @ )

    A Helm chart to manage serviceAccounts, roles and rolebindings
